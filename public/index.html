agent-pet

Tiny animated companion-pet widget. Self-hostable, no backend, ~14KB gzip.

One-line install

<script src="https://agent-pet.pages.dev/agent-pet-widget.iife.js"></script>

Then drive it:

AgentPet.setState('thinking');
AgentPet.say('Build done!', { link: '/results' });
AgentPet.configure({ name: 'Rex', glyph: '🦖' });

Try it

The pet you can see was loaded by the script tag at the bottom of this page.
