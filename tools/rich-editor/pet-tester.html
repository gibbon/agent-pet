Sample page content

This is a normal-looking page where the agent-pet widget is embedded. The pet is the small character on screen — it tracks your scroll position, plays its idle animation, and reacts to the action buttons above. Drop the widget on any page exactly like this:

<script src="/dist/agent-pet-widget.iife.js"></script>
<script>AgentPet.loadManifest(yourManifest)</script>

The rich runtime is lazy-loaded only if the manifest has runtime: "rich", so atlas-only pets pay no extra cost.
